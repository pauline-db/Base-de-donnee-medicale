# Medical SQLite Database

## Description

This project contains a small SQLite database about medicine and health-related information.

The database is designed to organize and connect information about diseases, symptoms, medicines, side effects, and medical incompatibilities.
The idea is to have a structure for a potential database in which a patient could enter one for example his symptoms and get potential diseases, medicines etc... and vice versa.
